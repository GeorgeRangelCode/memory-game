# Memory Game

Un juego de memoria interactivo construido con React y Vite.

## Descripción

Este es un juego clásico de memoria donde los jugadores deben encontrar parejas de cartas coincidentes. El objetivo es encontrar todas las parejas en el menor número de movimientos posible.

## Características

- Interfaz intuitiva y responsiva
- Contador de movimientos
- Temporizador
- Efectos visuales suaves
- Diferentes niveles de dificultad

## Instalación

1. Clona el repositorio:
```bash
git clone https://github.com/GeorgeRangelCode/memory-game.git
```

2. Instala las dependencias:
```bash
cd memory-game
yarn install
```

3. Inicia el servidor de desarrollo:
```bash
yarn dev
```

## Tecnologías utilizadas

- **React** - Biblioteca de JavaScript para construir interfaces de usuario
- **Vite** - Herramienta de construcción rápida para desarrollo
- **CSS3** - Para estilos y animaciones
- **ESLint** - Para mantener la calidad del código
